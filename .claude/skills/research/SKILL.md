# 트렌드/데이터 리서치

## 입력
- `$ARGUMENTS`: 리서치 키워드

## 프로세스
1. WebSearch로 최신 정보 수집 (3회 이상 검색)
2. 신뢰도 분류 — Tier 1/2만 채택, Tier 3 제외
3. 구조화된 리서치 노트로 정리

## 출력 형식
```markdown
# 리서치: [키워드]
**수집일:** YYYY-MM-DD

## 핵심 요약
- (3줄 이내)

## 주요 발견
### 1. [주제]
- 내용 + 출처

### 2. [주제]
- 내용 + 출처

## 데이터/통계
| 지표 | 수치 | 출처 |
|------|------|------|

## 소스 목록
- [제목](URL) — Tier 1/2 표기
```

## 규칙
- Tier 3 소스 절대 사용 불가
- 2년 이상 된 데이터는 "(YYYY년 기준)" 표기
- 출처 없는 주장 포함 불가
