# 소스 신뢰도 정책

## Tier 1 — 인용 가능 (출처 표기 권장)
공식 문서, 학술 논문, 정부/공공기관 통계, 기업 IR/공시 자료

## Tier 2 — 출처 명시 필수
기술 블로그, 컨퍼런스 발표 자료, 언론 보도, GitHub 레포지토리

## Tier 3 — 사용 불가
출처 불명 데이터, 커뮤니티 루머, 생성 AI 출력물을 원출처로 인용

## 인용 형식
"Jekyll은 Liquid 템플릿 엔진을 사용한다 ([공식 문서](https://jekyllrb.com/docs/liquid/))."
- 링크 텍스트는 서술적으로 ("여기" 또는 "링크" 금지)
- 2년 이상 된 자료는 "(YYYY년 기준)" 표기
