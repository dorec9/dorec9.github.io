## 이도현

경영기획/IT기획 직무를 지향하는 기획자입니다.

데이터 기반 의사결정, 프로젝트 설계, AI 활용 업무 자동화에 관심을 갖고 있으며, 이 블로그에서는 실무 관점의 인사이트를 기록합니다.

### 블로그 카테고리

| 카테고리 | 내용 |
|---------|------|
| 프로젝트 회고 | 설계 판단 과정 중심의 프로젝트 복기 |
| 직무 인사이트 | 경영기획·IT기획 실무 관련 분석 |
| 데이터 분석/통계 | 데이터 기반 분석 사례와 방법론 |
| 트렌드 리서치 | 업계 동향과 기술 트렌드 정리 |
| 경영·경제 분석 | 경영/경제 이슈에 대한 실무적 해석 |
| 하네스 설계 일지 | AI 에이전트 블로그 자동화 설계 과정 |
